## Quick start
1. Clone the repo
   - `git clone https://github.com/fand1l/task_tracker.git && cd task_tracker`
2. Create a virtual environment
   - `python -m venv .venv`
   - Activate: `. .venv/bin/activate` (Linux/macOS) or `.venv\Scripts\activate` (Windows)
3. Install dependencies (if present)
   - `pip install -r requirements.txt`
4. Run the app
   - `python manage.py runserver`

Then open your browser at http://localhost:5000 or http://localhost:8000 depending on your setup.
